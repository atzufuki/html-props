# Installation

You can install @html-props/core via JSR or import it directly from a CDN.

## Using Deno

```bash
deno add @html-props/core
```

## Using npm

```bash
npx jsr add @html-props/core
```

## CDN (ES Modules)

```typescript
import { HTMLPropsMixin } from 'https://esm.sh/@html-props/core';
```
