# create-html-props

> Scaffolding CLI for html-props projects

## Usage

### With Deno (recommended)

```
deno run -A jsr:@html-props/create <project-name>
```

Or, if installed globally:

```
create-html-props <project-name>
```

## Example

```
deno run -A jsr:@html-props/create my-app
cd my-app
# Start hacking!
```
